# 🏛️ Portal Rasmi Arrahnu Prihatin
**Pembangunan laman web korporat rasmi untuk Arrahnu Prihatin.** Projek ini merangkumi pengurusan maklumat dinamik bagi 26 cawangan di seluruh negara, memfokuskan kepada ketelusan maklumat dan kemudahan pelanggan.

**Pembangun:** Aisyah (Intern Sistem & Data Entry)  
**Lokasi Pembangunan:** Arrahnu Prihatin Pandu Lalu Melaka Raya

---

## 📝 Log Pembangunan (Development Logbook)

### [20/04/2026] - Fasa Perancangan & Pemilihan Kerangka
- **Aktiviti:** Sesi konsultasi bersama CEO mengenai hala tuju visual portal.
- **Tindakan:** - Mengenal pasti keperluan utama portal (maklumat cawangan, harga emas semasa, & hubungi kami).
  - Pemilihan template asas daripada **HTML5 UP (Solid State)** sebagai kerangka struktur.
  - Memulakan pembersihan kod asal template untuk disesuaikan dengan keperluan Arrahnu.

### [21/04/2026] - Transformasi Identiti Visual (Branding)
- **Aktiviti:** Penyelarasan warna tema mengikut identiti korporat Arrahnu Prihatin.
- **Tindakan:** - Mengubah skema warna asal (Biru) kepada tona **Merah Korp_Arrahnu** dan **Emas Eksklusif**.
  - Mengemaskini fail CSS untuk memastikan konsistensi warna pada setiap elemen butang dan 'banner'.
  - Memasukkan logo rasmi `logo.png` ke dalam struktur pengepala (header).

### [22/04/2026] - Pengurusan Lesen & Legalitas
- **Aktiviti:** Menetapkan status hak cipta pembangunan portal.
- **Tindakan:** - Mengubah suai deskripsi lesen asal (MIT/CC) kepada **Proprietary License** (Hak Milik Terpelihara).
  - Memastikan kredit asal kepada HTML5 UP dikekalkan mengikut syarat penggunaan template.
  - Menetapkan notis hak cipta rasmi: `Copyright © 2026 Arrahnu Prihatin`.

### [23/04/2026] - Pengemasan Struktur & Footer Responsive
- **Aktiviti:** Pembangunan bahagian Footer dan pengoptimuman paparan mudah alih (mobile).
- **Tindakan:** - Membangunkan seksyen "Hubungi Kami" dengan maklumat cawangan Melaka Raya.
  - Menyelaraskan kedudukan logo supaya responsif (Tengah pada telefon, Kanan pada desktop).
  - Melakukan 'commit' berkala ke GitHub untuk pemantauan sejarah kod.

---

## 🛠️ Teknologi Digunakan
- **Bahasa:** HTML5, CSS3, JavaScript.
- **Tools:** VS Code, GitHub, Vercel Hosting.
- **Template Dasar:** Solid State by HTML5 UP.
