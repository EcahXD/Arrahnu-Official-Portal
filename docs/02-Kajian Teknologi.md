# Bab 2 Kajian Teknologi

## Pengenalan

Bab ini membincangkan kajian teknologi yang digunakan dalam pembangunan Portal Ar-Rahnu Prihatin. Pemilihan teknologi yang sesuai merupakan aspek penting bagi memastikan laman web yang dibangunkan mampu berfungsi dengan baik, mudah diselenggara, responsif serta memenuhi keperluan semasa organisasi. Oleh kerana projek ini memfokuskan kepada pembangunan laman web statik, teknologi yang dipilih menekankan kesederhanaan sistem, prestasi capaian yang pantas, kos pembangunan yang rendah serta kemudahan proses penerbitan ke pelayan web.

Portal Ar-Rahnu Prihatin dibangunkan menggunakan gabungan teknologi pembangunan web moden seperti HTML5, CSS3 dan JavaScript. Selain itu, projek ini turut menggunakan templat reka bentuk sedia ada daripada HTML5 UP bagi mempercepatkan proses pembangunan antaramuka serta memastikan hasil akhir mempunyai penampilan profesional dan mesra pengguna. Dari sudut pembangunan perisian, Visual Studio Code digunakan sebagai persekitaran pembangunan utama kerana menawarkan pelbagai fungsi yang membantu proses pengekodan dan ujian laman web secara langsung.

## Teknologi HTML5

HTML5 merupakan bahasa penanda standard yang digunakan untuk membina struktur utama sesebuah laman web. Teknologi ini digunakan secara meluas dalam pembangunan laman web moden kerana menyokong pelbagai elemen semantik seperti `header`, `section`, `article`, `nav` dan `footer`. Penggunaan elemen-elemen ini membantu menghasilkan kod yang lebih tersusun, mudah dibaca serta mesra enjin carian.

Dalam projek Portal Ar-Rahnu Prihatin, HTML5 digunakan bagi membina keseluruhan struktur halaman seperti laman utama, bahagian profil syarikat, senarai perkhidmatan, maklumat cawangan serta ruangan hubungan. Penggunaan HTML5 juga memastikan laman web boleh dipaparkan dengan baik pada pelayar moden seperti Google Chrome, Microsoft Edge dan Mozilla Firefox.

Selain itu, HTML5 sesuai digunakan dalam pembangunan laman web statik kerana ia ringan, stabil dan tidak memerlukan pangkalan data atau konfigurasi pelayan yang kompleks. Hal ini menjadikan HTML5 pilihan yang sesuai untuk projek portal korporat berskala sederhana seperti Ar-Rahnu Prihatin.

## Cascading Style Sheets (CSS3)

CSS3 digunakan untuk mengawal reka bentuk visual laman web seperti warna, susun atur, tipografi, animasi dan responsif paparan. Dengan penggunaan CSS3, portal yang dibangunkan dapat menampilkan identiti korporat yang lebih profesional dan menarik kepada pengguna.

Dalam projek ini, CSS3 digunakan untuk menyesuaikan warna tema, susun atur kandungan, butang navigasi, imej latar belakang serta elemen antaramuka lain agar selari dengan identiti Ar-Rahnu Prihatin. Selain itu, ciri responsif dalam CSS3 membolehkan laman web dipaparkan dengan baik pada pelbagai saiz skrin termasuk komputer, tablet dan telefon pintar.

Penggunaan CSS3 juga membolehkan laman web mengekalkan penampilan yang konsisten pada setiap halaman serta meningkatkan pengalaman pengguna ketika melayari portal.

## JavaScript

JavaScript merupakan bahasa pengaturcaraan bahagian hadapan (front-end) yang digunakan untuk menambah interaktiviti pada laman web. Dalam pembangunan portal ini, JavaScript digunakan bagi fungsi asas seperti navigasi menu, animasi skrol, kesan visual serta peningkatan pengalaman pengguna ketika berinteraksi dengan laman web.

Walaupun portal ini dibangunkan secara statik, penggunaan JavaScript membantu menjadikan laman web lebih moden dan tidak terlalu bergantung kepada paparan kandungan statik semata-mata. Dengan gabungan JavaScript yang minimum dan terkawal, prestasi laman web masih dapat dikekalkan pada tahap optimum.

## Templat Solid State oleh HTML5 UP

Bagi mempercepatkan proses pembangunan serta memastikan reka bentuk laman web kelihatan profesional, projek ini menggunakan templat **Solid State** yang disediakan oleh HTML5 UP. Templat ini dipilih kerana mempunyai reka bentuk moden, kemas, responsif dan sesuai digunakan untuk laman web korporat.

Solid State menyediakan struktur halaman yang tersusun, animasi yang ringan serta susun atur kandungan yang menarik. Templat ini juga dibina menggunakan HTML5 dan CSS3 yang bersih, sekali gus memudahkan proses pengubahsuaian mengikut keperluan projek.

Dalam projek Portal Ar-Rahnu Prihatin, templat ini telah disesuaikan dari segi kandungan, warna, imej, maklumat syarikat dan navigasi supaya memenuhi identiti serta objektif portal rasmi syarikat. Penggunaan templat sedia ada juga menjimatkan masa pembangunan kerana pembangun tidak perlu membina antaramuka daripada kosong.

## Perisian Pembangunan: Visual Studio Code

Visual Studio Code (VS Code) digunakan sebagai perisian utama sepanjang proses pembangunan laman web. VS Code merupakan editor kod sumber yang popular kerana ringan, pantas serta menyokong pelbagai bahasa pengaturcaraan dan sambungan tambahan (extensions).

Antara kelebihan penggunaan VS Code dalam projek ini ialah kemudahan penyuntingan kod HTML, CSS dan JavaScript dalam satu platform. Selain itu, pelbagai sambungan tambahan digunakan bagi membantu meningkatkan produktiviti seperti fungsi pratonton langsung (live preview/live server), penformatan kod automatik, cadangan sintaks pintar serta pengurusan fail projek yang lebih teratur.

Fungsi pratonton langsung amat membantu kerana sebarang perubahan pada kod boleh dilihat serta-merta pada pelayar web tanpa perlu memuat semula halaman secara manual. Ini mempercepatkan proses reka bentuk, pengujian dan penambahbaikan antaramuka portal.

## Platform Pengehosan dan Domain

Bagi tujuan pembangunan dan ujian semasa projek dilaksanakan, portal ini dihoskan menggunakan domain percuma yang disediakan melalui platform GitHub Pages. Platform ini sesuai digunakan untuk laman web statik kerana mudah digunakan, stabil serta menyokong proses penerbitan terus daripada repositori projek.

GitHub Pages membolehkan laman web dipaparkan secara dalam talian untuk tujuan demonstrasi, semakan dan pengujian fungsi sebelum diserahkan kepada pihak syarikat. Selain itu, penggunaan platform ini tidak memerlukan kos tambahan sepanjang tempoh pembangunan projek.

Untuk penggunaan sebenar pada masa hadapan, portal ini dirancang untuk dimuat naik ke domain rasmi milik syarikat bagi meningkatkan imej profesional, keyakinan pelanggan serta identiti korporat Ar-Rahnu Prihatin. Penggunaan domain rasmi akan memberi nilai tambah kerana pelanggan lebih mudah mengenal pasti portal yang sah dan dipercayai.

## Kelebihan Teknologi yang Dipilih

Gabungan HTML5, CSS3, JavaScript, templat Solid State, VS Code dan GitHub Pages memberikan beberapa kelebihan kepada projek ini. Antaranya ialah kos pembangunan yang rendah, proses pembangunan yang lebih pantas, laman web ringan dan cepat dimuatkan, mudah diselenggara serta sesuai untuk projek portal statik.

Selain itu, teknologi yang dipilih juga membolehkan portal dibangunkan dengan penampilan profesional tanpa memerlukan sistem pangkalan data atau pelayan yang kompleks. Ini sangat sesuai dengan keperluan semasa organisasi yang memerlukan laman web rasmi sebagai medium penyampaian maklumat korporat.

## Kesimpulan

Secara keseluruhannya, pemilihan teknologi dalam pembangunan Portal Ar-Rahnu Prihatin dibuat berdasarkan faktor kesesuaian, kos, kemudahan pembangunan dan keberkesanan sistem. Penggunaan HTML5, CSS3 dan JavaScript sebagai asas pembangunan laman web, bersama templat Solid State daripada HTML5 UP, berjaya menghasilkan portal statik yang moden dan responsif.

Di samping itu, penggunaan Visual Studio Code serta platform GitHub Pages membantu memudahkan proses pembangunan, pengujian dan penerbitan laman web. Dengan teknologi yang dipilih ini, portal yang dibangunkan mampu memenuhi objektif projek serta menjadi asas kepada penambahbaikan digital syarikat pada masa hadapan.
